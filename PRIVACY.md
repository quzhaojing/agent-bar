# Privacy Policy for Agent Bar

**Last Updated:** November 21, 2025
**Effective Date:** November 21, 2025

## Overview

Agent Bar ("the Extension") is a browser extension that provides AI-powered text processing capabilities. This Privacy Policy explains how we collect, use, store, and protect your information when you use the Agent Bar Chrome extension.

## Information We Collect

### 1. Information You Provide

- **API Keys**: We store your AI service provider API keys (OpenAI, Claude, Gemini, etc.) locally in your browser
- **Extension Settings**: Toolbar configurations, custom prompts, and user preferences
- **Website Patterns**: URL matching rules for customizing toolbar behavior

### 2. Automatically Collected Information

- **Selected Text**: Text content you intentionally select for AI processing
- **Current Website URL**: To provide context-appropriate toolbar options
- **Extension Usage Data**: Local storage of your AI request history for convenience

## How We Use Your Information

- **To Provide Core Functionality**: Process selected text using your configured AI providers
- **To Store User Preferences**: Save your settings, toolbar configurations, and API keys locally
- **To Improve User Experience**: Provide relevant toolbar options based on current website context
- **For Extension Operation**: Maintain local history of AI responses for your reference

## Information Storage and Security

### Local Storage Only

- All user data is stored **locally in your browser** using Chrome's storage API
- **No data is transmitted to our servers** or any third-party services
- API keys and personal configurations never leave your browser

### Data Security

- API keys are stored securely in browser local storage
- Extension data is encrypted at rest using Chrome's built-in security
- We implement industry-standard security practices for browser extensions

## Third-Party Services

### AI Providers

When you use Agent Bar to process text:

- Selected text is sent directly to your configured AI provider (OpenAI, Claude, etc.)
- This communication occurs between your browser and the AI provider only
- We do not intercept, store, or access these communications

### Example:

- Selected text → Your Browser → AI Provider → AI Response → Your Browser

## Data We Don't Collect

We **DO NOT** collect:

- Personal identification information
- Browsing history or website content beyond selected text
- User analytics or tracking data
- Information about your browsing habits
- Data from websites where you don't use the extension

## User Rights and Control

### Your Data, Your Control

- **Access**: View all stored data in extension settings
- **Modification**: Update your configurations, API keys, and preferences anytime
- **Deletion**: Remove all stored data by uninstalling the extension
- **Export**: Export your toolbar configurations and settings

### Data Retention

- Data remains stored locally until you delete it or uninstall the extension
- No automatic data deletion or expiration policies
- You control how long your data is stored

## Children's Privacy

Agent Bar is not directed to children under 13 years of age. We do not knowingly collect personal information from children under 13. If we become aware that we have collected information from a child under 13, we will delete it immediately.

## Changes to This Privacy Policy

We may update this Privacy Policy from time to time to reflect changes in our practices or for other operational, legal, or regulatory reasons. We will post the updated policy in the extension and update the "Last Updated" date.
